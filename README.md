# WINDOWS95-0
Clone de Windows 95 en Pascal (Turbo Pascal ou Free Pascal). Le projet se veut plus éducatif qu'autre chose, il n'a pas la prétention de vouloir remplacer Windows 95.

<h3>Version 0.1</h3>

Pour cette version 0.1, il offre la possibilité de changer, d'arrêter, redémarrer ou fermer l'ordinateur. Un support minimum de la souris et du clavier.
Possibilité de visualiser l'heure et d'exécuter des commandes.
